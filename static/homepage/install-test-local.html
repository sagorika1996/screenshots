Dev Firefox Screenshots

This installs a copy of the add-on that points at this instance of the Screenshots server.
To use this version of the add-on you must:
  1. Use Firefox Nightly
  2. Open about:config and set:
    • xpinstall.signatures.required to false
    • extensions.screenshots.system-disabled to false
  3. Click Install
  4. You may have to restart the browser. If shots are uploaded to screeshots.firefox.com instead of this server, then a restart is required.
  5. You must use a localized build of Nightly to view the localized add-on.
  6. To see information about this server, see __version__
